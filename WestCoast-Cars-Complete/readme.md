# WestCoast-Starter projekt

Detta är ett startprojekt för hur vi ska organisera och strukturera en JavaScript applikation.
I projektet finns ingen egentlig struktur förutom att vi har mappar för assets och lib.

![Starter Structure](https://res.cloudinary.com/softtech-dev/image/upload/v1769086025/westcoastcars-starter-structure_uyezuo.png)

### Komma igång

Ladda hem projektet antingen genom att göra en klon eller ta hem projektet som en zip fil och expandera zip-filen.
Öppna upp projektet i t ex VS Code.
Öppna en bash eller zsh terminal i VS Code och kör följande kommando i terminalen:
`npm install` eller `npm i` och tryck på Enter.
För att starta applikationen skriv följande kommando i terminalfönstret:
`npm run dev`
Nu kommer json-server att starta vårt rest api och applikationen kommer att starta upp på adressen:
_http://localhost:8080._
